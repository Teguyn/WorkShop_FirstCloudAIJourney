This blog post shares the implementation of a multi-layered security model (Defense in Depth) on AWS to protect a Serverless Generative AI application, including edge protection (WAF & CloudFront), user authentication and API protection (Cognito & API Gateway), least-privilege IAM roles, and automated monitoring.

### [Blog 3 - How One Organization Cut AWS Costs by 39% in 12 Weeks](3.3-Blog3/)
This blog post analyzes the 3-phase cloud cost optimization strategy of a SaaS organization, from upgrading EBS gp2 to gp3, configuring S3 VPC Gateway Endpoints to reduce NAT Gateway charges, to consolidating Load Balancers and leveraging Karpenter to optimize EKS compute nodes using AWS Graviton ARM processors.
